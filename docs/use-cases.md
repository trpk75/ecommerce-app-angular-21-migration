# Use Cases

## UC-01 Browse Products

Actor: Shopper

Goal: View available products with product name, weight, price, and image.

Main flow:
1. Shopper opens the products page.
2. Application displays the product grid.
3. Shopper scans product cards.

Success outcome: Products are visible and ready for cart actions.

## UC-02 Search Products

Actor: Shopper

Goal: Find a product by entering text in the search box.

Main flow:
1. Shopper types a product name fragment.
2. Application filters the list by product name.
3. Shopper sees matching products.

Success outcome: Only matching products are shown.

## UC-03 Sort Products

Actor: Shopper

Goal: Sort products by name or price.

Main flow:
1. Shopper selects a sort option.
2. Application reorders the visible product list.

Success outcome: Products are ordered according to the selected criterion.

## UC-04 Manage Cart

Actor: Shopper

Goal: Add items, change quantities, and empty the cart.

Main flow:
1. Shopper adds a product to the cart.
2. Application updates cart quantity and total.
3. Shopper increments, decrements, replaces, or clears quantities.

Success outcome: Cart items and totals remain accurate.

## UC-05 Checkout

Actor: Shopper

Goal: Provide billing information and place an order.

Main flow:
1. Shopper opens billing.
2. Shopper enters required billing fields.
3. Application validates the form.
4. Shopper confirms the order.
5. Application displays an invoice.

Success outcome: Order confirmation and invoice are displayed.
