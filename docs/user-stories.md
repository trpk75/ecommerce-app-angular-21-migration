# User Stories

- As a shopper, I want to browse all available products so I can decide what to buy.
- As a shopper, I want to search products by name so I can quickly find an item.
- As a shopper, I want to sort products by name or price so I can compare options.
- As a shopper, I want to add products to a cart so I can purchase multiple items together.
- As a shopper, I want to adjust item quantities so my cart reflects what I intend to buy.
- As a shopper, I want to clear my cart so I can restart my shopping session.
- As a shopper, I want form validation during billing so I know which details are required.
- As a shopper, I want an order invoice so I have confirmation of the placed order.
- As a maintainer, I want the Angular 6 source retained so I can compare legacy and migrated behavior.
- As a maintainer, I want unit tests around cart and product list logic so migration regressions are easier to catch.
